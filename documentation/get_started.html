2. File structure

3. What's included

Theme Venera comes equipped with HTML, CSS, and JS for all sorts of things, but they can be summarized with a handful of categories visible at the top of the Theme Venera documentation.

Files Contents

  • index.html - markup for homepage
  • contact.html - contact page with contact form and map
  • create_account.html - create account example page
  • html_blog/single_post.html - markup for blog page with a single post and a side bar
  • html_blog/two_posts_per_line.html - markup for blog page list of posts, two per line and a side bar
  • html_blog/one_post_per_line.html - markup for blog page list of posts, one per line and a side bar
  • html_features/pricing_table.html - pricing table example
  • html_features/basic_elements.html - markup for most of the elements like testimonials, recent work, progress bars, alerts, forms, tables, tabs and pills, tooltips and many more
  • html_features/awesome_icons.html - list of font awesome icons with classes related to each icon
  • html_features/about_us_page.html - markup for about us page
  • html_features/coming_soon_page.html - markup for Coming Soon Page/Under Maintanance
  • html_features/gallery_of_images.html - markup for gallery of images

Docs sections

Scaffolding

Global styles for the body to reset type and background, link styles, grid system, and two simple layouts.

Base CSS

Styles for common HTML elements like typography, code, tables, forms, and buttons. Also includes Font Awesome, a great little font based icon set.

Components

Basic styles for common interface components like tabs and pills, navbar, alerts, page headers, breadcrumbs, testimonials and more.

Extra Components

Extra components for custom interface like pricing tables, image galleries, page not found, coming soon page and more.

Blog

Styles for blog related pages.

JavaScript plugins

Similar to Components, these JavaScript plugins are interactive components for things like tooltips, popovers, modals, and more.

4. Basic HTML template

With a brief intro into the contents out of the way, we can focus on putting Theme Venera to use. To do that, we'll utilize a basic HTML template that includes everything we mentioned in the File structure.

Now, here's a look at a typical HTML file:

  <!DOCTYPE html>
  <html>
  <head>
  <title>Theme Venera Template</title>
  </head>
  <body>
  <h1>Hello, world!</h1>
  <script src="//code.jquery.com/jquery-latest.js"></script>
  </body>
  </html>

To make this a Theme Venera template, just include the appropriate CSS and JS files:

<!DOCTYPE html>
<html>
<head>
<title>Theme Venera Template</title>
<!-- Add Theme Venera stylesheet -->
<link href="css/theme_venera.css" rel="stylesheet" media="screen">
<!-- Add Google Web Fonts support -->
<link href="//fonts.googleapis.com/css?family=Abel:400|Oswald:300,400,700" media="all" rel="stylesheet" type="text/css" >
</head>
<body>
<h1>Hello, world!</h1>
<script src="js/jquery-1.10.1.min.js"></script>
<script src="js/theme_venera.min.js"></script>
</body>
</html>

And you're set! With those two files added, you can begin to develop any site or application with Theme Venera.

What if I want to use a blue color scheme?

It's super easy. Just include a theme_venera_blue.css (contact me if you can not find this file) file instead of theme_venera.css in your html like so:

<!DOCTYPE html>
<html>
  <head>
    <title>Theme Venera Template</title>
    <!-- Add Theme Venera BLUE stylesheet -->
    <link href="css/theme_venera_blue.css" rel="stylesheet" media="screen">
    <!-- Add Google Web Fonts support -->
    <link href="//fonts.googleapis.com/css?family=Oswald:400,300,700" media="all" rel="stylesheet" type="text/css" >
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="js/jquery-1.10.1.min.js"></script>
    <script src="js/theme_venera.min.js"></script>
  </body>
</html>
